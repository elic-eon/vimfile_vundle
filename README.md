### vimfile vundle

#### Install
1. ``` git clone https://github.com/elic-eon/vimfile_vundle.git ```  
2. ``` ln -s vimfile_vundle ~/.vim ```
3. ``` ln -s ~/.vim/vimrc ~/.vimrc ```
4. ``` git clone https://github.com/gmarik/Vundle.vim.git ~/.vim/bundle/Vundle.vim ```
5. # ``` vim ```
6. ``` :VundleInstall ```
